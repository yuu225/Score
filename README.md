# 学生成绩管理系统

基于 Vue 3 + TypeScript + Element Plus 开发的学生成绩管理系统。

## 功能特点

- 用户认证与授权
- 成绩录入与查询
- 学生信息管理
- 课程管理
- 成绩统计分析
- 个人信息管理

## 技术栈

- Vue 3
- TypeScript
- Vite
- Element Plus
- Vue Router
- Pinia
- LocalStorage
